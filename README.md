# Frankie-AI
A locally hosted AI chatbot built to allow users to utilize multiple models, including Llama 3.1, Phi 3, Mistral, Gemma 2, and other models. It runs entirely without an internet connection, featuring an intuitive interface and supporting diverse use cases.
